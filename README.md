# HRSync – SaaS Landing Page

## Overview
HRSync is a modern and responsive SaaS landing page designed for a Human Resource Management platform. The project was developed as part of the TEYZIX CORE Frontend Web Development Internship using HTML, CSS, JavaScript, and Bootstrap 5.

## Features
- Responsive design for desktop, tablet, and mobile devices
- Modern and clean user interface
- Hero section with call-to-action
- Features section
- Pricing section
- Testimonials section
- Contact section
- Smooth navigation
- Bootstrap 5 components

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
